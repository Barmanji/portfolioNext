## Overview
**Password Generator** is a react based webapp that i made while learning basics, like how to handle context API etc.


### Description

This password generator app provides a quick and secure way to create strong passwords. Built with a focus on user customization, the app allows users to select password length and include special characters for enhanced security. With each generation, the app provides unique, randomized passwords to suit various needs.

**Background:** The app features a dynamic, customizable background that can change based on user preferences. Its clean design and subtle animations make it both a practical tool for security and a visually engaging experience, perfect for users looking for strong password **need numbers? symbols? fix length?
